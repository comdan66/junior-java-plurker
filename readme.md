# Welcome To OA's 大三作品 Java Plurker!
此作品是當年大學三年級時期藉由噗浪官方提供的 API、SDK 所實作的 Java GUI 程式，實作一套結合噗浪微網誌的社群管理軟體。此次功能包含了軟體安裝、多角色登入、發訊息、回復訊息、智慧回復機器人..等。
